## 링크 Links

- 외부 링크 External Links

  - 인라인 링크

  [Google](https://www.google.com '구글')

  ```markdown
  [Google](https://www.google.com '구글')
  ```

  - 참조 링크

  [Google][1]
  [Naver][2]

  ```markdown
  [Google][1]
  [Naver][2]

  [1]: https://www.google.com/
  [2]: https://www.naver.com/
  ```

  - URL 링크

  <https://www.google.com/>

  ```markdown
  <https://www.google.com/>
  ```

- 내부 링크 Internal (Anchored) Links

[H1](#h1)

```markdown
[H1](#h1)
```

[1]: https://www.google.com/
[2]: https://www.naver.com/

## 리스트 Lists

- 순서 있는 리스트 Ordered Lists

  1. item 1
  2. item 2
  3. item 3

  ```markdown
  1. item 1
  2. item 2
  3. item 3
  ```

- 순서 없는 리스트 Unordered Lists

  - item 1
    - item 1-1
    - item 1-2

  ```markdown
  - item 1
    - item 1-1
    - item 1-2

  * item 1
    - item 1-1
    - item 1-2

  - item 1
    - item 1-1
    - item 1-2
  ```
